# POS_by_voice
음성으로 합계를 계산해주는 앱

음성 입력 방법:
1. 마이크 버튼을 켜고 자유롭게 말씀하세요.
2. 예: "박카스 6500원 3개, 민영이 100600원, 7000원"
3. 예: "600원 더하기 18950원 더하기 20000원" (각각 항목으로 분리됩니다.)

수정 및 관리:
1. 수정 명령: "박카스를 쌍화탕으로 수정해줘". "3개 말고 5개"
2. 초기화: "초기화 해줘"
3. 할인: 항목의 합계금액은 클릭하거나 하단의 총 합계를 클릭하여 10% 또는 직접 할인을 지정할 수 있습니다.

과거 기록 및 탭 저장:
1. 결제 1, 2, 3 탭은 화면을 새로고침해도 그대로 유지됩니다.
2. 계산 완료 및 저장을 누르면 과거 기록에 저장되어 언제든 다시 불러올 수 있습니다.
